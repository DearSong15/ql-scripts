# QL Scripts - 青龙面板脚本库

适用于青龙面板的 JS 脚本合集。

## 脚本列表

| 文件名 | 原名 | 说明 |
|--------|------|------|
| `ks_yanghao_v1.js` | ks养号 | 快手平台养号脚本 v1（本地接口版） |
| `zsp_video.js` | 中视频 | 中视频平台脚本 |
| `ks_jisu_v2.js` | 脚本库快手极速版 | 快手极速版脚本 v2（完整功能版） |

## 青龙面板拉取方式

在青龙面板 → 订阅管理 → 添加订阅，填写：

```
https://github.com/DearSong15/ql-scripts.git
```

或者在定时任务中执行拉取命令：

```bash
ql repo https://github.com/DearSong15/ql-scripts.git
```

## 注意事项

- 请确保青龙面板可以访问 GitHub（或配置代理）
- 脚本运行需要配置相应的环境变量
